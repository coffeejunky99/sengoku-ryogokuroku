## 1. 位置付け

本書は、縦切り試作の全国マップに使用する4勢力、10城、13街道、初期所有勢力および論理座標系を定義するGDD付属の正式仕様である。

- Git管理上の正本は本Markdown版とする。
- GDD本文と本書が矛盾する場合はGDD本文を優先する。
- DOCX版は人間向けの配布・レビュー用スナップショットであり、Markdown版と異なる場合はMarkdown版を優先する。
- 本書にない人口、農業、商業、防御、施設、武将、AI、移動時間、移動補正、補給などの値は確定しない。

## 6. Phase02表示用定義

Phase02は完全な`ClanDefinition`、`CastleDefinition`、`RouteDefinition`を要求せず、次の表示用定義だけを使用する。

```ts
interface MapClanDefinition {
  readonly id: ClanId;
  readonly name: string;
  readonly color: HexColor;
  readonly crestAssetKey: string;
  readonly identificationSymbol: string;
  readonly capitalCastleId: CastleId;
  readonly playable: boolean;
}

interface MapCastleDefinition {
  readonly id: CastleId;
  readonly name: string;
  readonly region: string;
  readonly position: MapPosition;
  readonly initialOwnerClanId: ClanId;
}

interface MapRouteDefinition {
  readonly id: RouteId;
  readonly fromCastleId: CastleId;
  readonly toCastleId: CastleId;
  readonly distance: number;
  readonly terrain: "mountain" | "plain";
  readonly isBidirectional: true;
}

interface MapDefinition {
  readonly logicalWidth: number;
  readonly logicalHeight: number;
  readonly origin: "top-left";
  readonly xAxis: "positive-right";
  readonly yAxis: "positive-down";
  readonly coordinateUnit: "logical-pixel";
  readonly contentPadding: number;
  readonly initialCameraCenter: MapPosition;
  readonly initialZoom: number;
  readonly minimumZoom: number;
  readonly maximumZoom: number;
  readonly clans: readonly MapClanDefinition[];
  readonly castles: readonly MapCastleDefinition[];
  readonly routes: readonly MapRouteDefinition[];
}
```

## 7. 整合性条件

- 勢力ID、城ID、街道IDは各カテゴリ内で一意であること。
- 全本拠地、初期所有勢力、街道端点の参照先が存在すること。
- 各勢力に本拠地が1つ存在すること。
- 全城座標がコンテンツ領域内にあること。
- 街道に自己接続および同一端点ペアの重複がないこと。
- 街道グラフが連結していること。
- 全勢力に色以外の識別方法が存在すること。
